# Trivias

Proyecto web estático (HTML/CSS/JS) sin dependencias.

## Janus Governance Layer

Este repositorio es un caso real de implementación de Janus como capa de gobernanza sobre un producto vivo (la app de trivias se mantiene intacta), incorporando estructura mínima para trazabilidad y validación sin reemplazar el código existente; ver `docs/` para protocolos/casos y `demos/` para escenarios de verificación.
